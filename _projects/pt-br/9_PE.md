Concordo com o [Evan Chen](https://web.evanchen.cc/techspeech.html) nesta. Por favor, aprenda a programar.

O Project Euler não é o melhor recurso para aprender programação a partir de primeiros princípios, mas sim para despertar sua curiosidade pelo pensamento algorítmico. Aqui está um problema [legal](https://projecteuler.net/problem=216) para pensar:

<p style="text-align: center;">
Quantos números da forma $2n^2-1$ são primos, para $1 \leq n \leq 50{,}000{,}000$?
</p>
